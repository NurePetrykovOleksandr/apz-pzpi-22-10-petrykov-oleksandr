Опис прийнятих інженерних рішень
Проект призначений для моніторингу вологості ґрунту та інших параметрів навколишнього середовища з використанням IoT-сенсорів. Основні інженерні рішення:
Використано Flutter як основу для універсального доступу: Android/iOS/Web.
Для веб-застосунку використання flutter_web з адаптацією під адміністрування.
Локалізація реалізується через пакет flutter_localizations.
Взаємодія компонентів:
UI екрани спілкуються з сервісами через API (services/).
Дані (вологості, погоди) відображаються через моделі та графіки.
Web доступ забезпечується через Flutter web build (index.html, manifest.json).

Висновки
Розроблено front-end частину системи з окремими інтерфейсами для користувачів і адміністраторів. Забезпечено підтримку локалізації та інтернаціоналізації, включно з перемиканням мов, форматами дат і напрямками тексту. Реалізовано основні функції адміністрування: керування користувачами, налаштування, імпорт/експорт та резервне копіювання даних.
